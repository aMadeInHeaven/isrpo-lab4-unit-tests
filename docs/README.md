># Geometric lib

**`Geometric Lib`** - это библиотека на Python для вычисления площадей и геометрических фигур. Библиотека состоит из четырех модулей, каждый из которых отвечает за работу с определенной фигурой:

* `rectangle.py` - прямоугольники
* `square.py` - квадраты
* `triangle.py` - треугольники
* `circle.py` - круги
  


## Описание функций
### Модуль: `rectangle.py`
- **`area(length, height)`**:
```
Вычисляет площадь прямоугольника по его длине и высоте.

Для расчета используется формула: S = length * height.

Параметры:
        length (число): длина прямоугольника
        height  (число): высота прямоугольника

Возвращаемое значение:
        area_of_rectangle (число): площадь прямоугольника

Пример:
    length = 3
    height = 4
    area(length, height) # 3 * 4 = 12
```

- **`perimeter(length, height)`**:
```
Вычисляет периметр прямоугольника по его длине и высоте.

Для расчета используется формула: P = 2 * (length + height)

Параметры:
        length (число): длина прямоугольника
        height  (число): высота прямоугольника

Возвращаемое значение:
        perimeter_of_rectangle (число): периметр прямоугольника

Пример:
    length = 3
    height = 4
    perimeter(length, height) # 2 * (3 + 4) = 14
```

### Модуль: `square.py`
- **`area(a)`**:
```
Вычисляет площадь квадрата по длине его стороны.

Для расчета используется формула: S = a * a

Параметры:
        a (число): длина стороны квадрата

Возвращаемое значение:
        area_of_square (число): площадь квадрата

Пример:
    a = 4
    area(a) # 4 * 4 = 16
```

- **`perimeter(a)`**:
```
Вычисляет периметр квадрата по длине его стороны.

Для расчета используется формула: P = 4 * a

Параметры:
        a (число): длина стороны квадрата

Возвращаемое значение:
        perimeter_of_square (число): периметр квадрата

Пример:
    a = 5
    perimeter(a) # 4 * 5 = 20
```


### Модуль: `triangle.py`
- **`area(a, height)`**:
```
Вычисляет площадь треугольника по длине его основания и высоты, проведенной к этому основанию.

Для расчета используется формула: S = a * height / 2

Параметры:
        a (число): длина основания треугольника
        height (число): длина высоты трегольника

Возвращаемое значение:
        area_of_triangle (число): площадь треугольника

Пример:
    a = 4
    height = 10
    area(a, height) # 4 * 10 / 2 = 20
```

- **`perimeter(a, b, c)`**:
```
Вычисляет периметр треугольника по длинам трех его сторон.

Для расчета используется формула: P = a + b + c

Параметры:
        a (число): длина первой стороны треугольника
        b (число): длина второй стороны треугольника
        c (число): длина третьей стороны треугольника

Возвращаемое значение:
        perimeter_of_triangle (число): периметр треугольника

Пример:
    a = 3
    b = 4
    c = 5
    perimeter(a, b, c) # 3 + 4 + 5 = 12
```


### Модуль: `circle.py`
- **`area(r)`**:
```
Вычисляет площадь круга по длине его радиуса.

Для расчета используется формула: S = pi * r * r

Параметры:
        r (число): длина радиуса круга

Возвращаемое значение:
        area_of_circle (число): площадь круга

Пример:
    r = 10
    area(r) # pi * 10 * 10 ~ 314
```

- **`perimeter(r)`**:
```
Вычисляет длину окружности по длине ее радиуса.

Для расчета используется формула: P = 2 * pi * r

Параметры:
        r (число): длина радиуса окружности

Возвращаемое значение:
        len_of_circumference (число): длина окружности

Пример:
    r = 10
    perimetr(r) # 2 * pi * 10  ~ 62.8
```

## Использование

```python
# Пример импорта
from rectangle import area

# Расчет площади прямоугольника
result = area(7, 3)  # 21
```
